Welcome to React Khatra Placeholder

A react component library for creating skeleton placeholders of your need.

Installation​

  • npm
  • yarn
  • pnpm
npm install react-khatra-placeholder
yarn add react-khatra-placeholder
pnpm add react-khatra-placeholder

Requirement​

  • React.js version 16 or above
